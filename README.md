# CONTROL-ESCOLAR

Aplicación simple para registrar y gestionar maestros (CRUD) usando Node.js y Express.

Instalación y ejecución:

```bash
npm install
npm start
# abrir http://localhost:3000
```

Archivos principales:
- `controlMaestros.js`: servidor y API REST
- `data/maestros.json`: almacenamiento simple en JSON
- `public/`: frontend (index.html, app.js, style.css)
